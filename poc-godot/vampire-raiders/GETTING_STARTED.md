# Getting Started - Vampire Raiders Prototype

## What's Been Created

### Player Movement ✅
- **Scene:** `scenes/player/Player.tscn`
- **Script:** `scripts/player/player.gd`
- **Features:**
  - 8-directional movement
  - Speed: 300 units/sec (adjustable in inspector)
  - Camera follows player

### Game World ✅
- **Scene:** `scenes/world/GameWorld.tscn`
- **Features:**
  - Simple dark background
  - Player spawns at center

## Controls

- **WASD** or **Arrow Keys** - Move player

## How to Test

1. Open the project in Godot 4
2. Press **F5** to run the game
3. Use WASD or arrow keys to move around
